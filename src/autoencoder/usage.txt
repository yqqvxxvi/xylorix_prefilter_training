The script now uses the combined loss by default with these weights:
  - MSE: 1.0 (base)
  - Perceptual: 0.1
  - Gradient: 0.05

  Default usage:
  python scripts/train_oneclass_conv_autoencoder.py \
      --endgrain_dir /path/to/endgrain \
      --world_dir /path/to/world \
      --output_dir results/conv_autoencoder \
      --epochs 100

  Custom loss weights:
  python scripts/train_oneclass_conv_autoencoder.py \
      --endgrain_dir /path/to/endgrain \
      --perceptual_weight 0.2 \
      --gradient_weight 0.1 \
      --epochs 100

  Disable perceptual loss (only MSE + gradient):
  python scripts/train_oneclass_conv_autoencoder.py \
      --endgrain_dir /path/to/endgrain \
      --disable_perceptual \
      --epochs 100
